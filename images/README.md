DeathTG image contract

Put your PNG files in `images/` or `Image/` with these exact names:

- `DeathTG_welcome.png` - welcome card sent after first successful setup
- `DeathTG_update_available.png` - update notification card sent in Telegram
- `DeathTG_creating_backup.png` - backup flow card

Module images:

- Put per-module fallbacks in `images/modules/`
- Shared fallback image: `images/modules/Module.png`
- Optional per-module file names: `MyModule.png`, `mymodule.png`, or another normalized module name match

Local module folders:

- `modules/MyModule/Module.png`
- `modules/MyModule/MyModule.py` or `modules/MyModule/main.py` or `modules/MyModule/__init__.py`

Recommended aspect ratio for module cards:

- `16:9`

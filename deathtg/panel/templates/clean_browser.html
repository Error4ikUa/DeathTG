Module contract

Folder modules are supported

Local module can be a single .py file or a folder like modules/MyModule/ with MyModule.py, main.py or __init__.py plus optional Module.png (16:9).

If a module has no own image, DeathTG falls back to images/modules/<module_name>.png or shared images/modules/Module.png.
